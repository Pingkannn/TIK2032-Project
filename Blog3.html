HTML

HTML telah dianggap sebagai standar pemrograman untuk membuat website resmi. Lebih dari 92,3% website di dunia menggunakan HTML sebagai bahasa pemrogramannya. Ingin mempelajari dasar-dasar HTML untuk membuat website perusahaan atau e-commerce. Alangkah baiknya Anda memahami terlebih dahulu tentang pengertian HTML, komponen, fungsi, dan cara kerjanya sebelum mempraktikkannya dalam pembuatan website.

  1. Pengertian HTML

    2. HTML adalah Bahasa Standar Pemrograman.

    HTML alias Hypertext Markup Language merupakan bahasa markup yang digunakan untuk membuat halaman website dan aplikasi web. HTML berisi sejumlah kode yang berguna sebagai struktur sebuah website. Kode tersebut berisi kombinasi teks dan simbol yang dibuat berdasarkan format standar kode internasional atau ASCII (American Standard Code for Information Interchange).

    Anda bisa membuat heading, gambar, memasukkan tautan, dan komponen lainnya dalam halaman website menggunakan HTML. Struktur kodenya pun sederhana, yaitu terdiri dari tag dan atribut sebagai markup pada halaman website.

  2. Sejarah dan Perkembangan HTML

    3. Popularitas HTML yang meningkat tentu tak lepas dari sejarah dan perkembangannya yang cukup panjang. Bagaimana awal mula bahasa pemrograman ini digunakan?

    HTML lahir pada tahun 1991 dan diciptakan oleh Tim Berners-Lee. Pada awalnya, HTML digunakan terbatas di kalangan para ilmuwan untuk mengakses dokumen. Setahun kemudian, HTML mulai digunakan untuk umum seiring perkembangan domain WWW (World Wide Web). HTML dan WWW sama-sama dimanfaatkan agar bisa menyebarkan informasi kepada banyak orang melalui Internet.

    Bahasa markup ini mengalami perkembangan yang signifikan hingga sekarang. Sampai saat ini, sudah ada enam versi HTML yang ditemukan dan digunakan dalam website. Versi HTML2 yang lahir pada tahun 1995 merupakan awal mula lahirnya website yang lebih interaktif.

  3. Komponen dalam HTML

    4. Ada tiga komponen utama dalam HTML. Tiga komponen tersebut saling berkaitan satu sama lain sehingga menyusun sebuah halaman web. Inilah komponen penting dalam HTML.

    • Tag:Pertama, ada komponen tag yang merupakan teks khusus untuk menandai suatu konten. Kode dalam tag menentukan tampilan konten berupa gambar, teks, atau video dalam aplikasi web. Strukturnya terdiri dari tag pembuka dan tag penutup dan diisi dengan konten di antara keduanya. Seperti ini contoh tag dalam HTML.
    • Elemen: Elemen merupakan seluruh kode HTML yang dimulai dari tag pembuka sampai tag penutup. Ada banyak jenis elemen yang digunakan dalam pembuatan website. Berdasarkan cara penulisannya, ada dua jenis elemen, yaitu elemen normal dan elemen tunggal atau elemen kosong.
      Ada pula jenis elemen yang dibagi berdasarkan tampilannya, yaitu block-level element dan in-line element. Jenis terakhir adalah elemen bersarang (nested element), yaitu elemen yang memiliki elemen lain di dalamnya. Anda bisa memahami lebih lanjut tentang jenis-jenis elemen HTML ketika belajar coding.
    • Atribut: Terakhir, ada komponen atribut yang merupakan informasi tambahan pada elemen dan ditulis pada bagian tag pembuka. Anda bisa menambahkan lebih dari satu atribut dalam sebuah elemen. Atribut ditulis dengan rumus nama_atribut=”value_atribut” dan bagian value atribut harus diapit dengan tanda petik ganda atau petik tunggal. Atribut juga terdiri dari dua jenis, yaitu atribut global dan atribut khusus. Atribut global bisa digunakan pada semua jenis tag atau elemen pada HTML, sedangkan atribut khusus hanya bisa digunakan pada tag yang didukung oleh atribut tersebut.

  4. Bagaimana Cara Kerja HTML?

    5. Mungkin Anda penasaran dengan cara kerja HTML hingga menjadi tampilan website yang menarik untuk dikunjungi. Proses kerjanya dimulai dari pembuatan dokumen HTML dengan aplikasi editor HTML yang tersedia. Anda harus menyimpan dokumen tersebut dengan ekstensi ‘.html’ atau ‘htm’.

    Oleh karena website terdiri dari banyak halaman, Anda perlu membuat lebih dari satu dokumen HTML. Setiap dokumennya bisa mewakili halaman website, seperti blog, homepage, tentang perusahaan, kontak, halaman produk, dan sebagainya. Dokumen tersebut akan menyusun bagian-bagian dalam halaman website, seperti header, footer, isi konten, paragraf, dan sebagainya. Namun, Anda hanya bisa membuka file HTML di browser.

    Halaman website pun siap diluncurkan setelah disusun dengan file HTML yang telah dibuat. Sebelum meluncurkannya, Anda harus menyewa layanan hosting website, kemudian mengunggah file website-nya terlebih dahulu. Browser akan melakukan finalisasi file HTML tersebut menjadi halaman website. Pengunjung website pun akan melihat tampilan akhir website dengan mudah dan nyaman setelah proses pembuatan HTML tersebut.
